# The Holomatic Principle

## Brief Definition

Roy Ascott's coinage for the holographic-network property whereby all data flowing through any access node are equally and simultaneously held in the network's memory, so that to be at any one interface is to be in the virtual presence of every other interface throughout the network.

## Longer Explanation

The holomatic principle names the way a telematic network behaves like a hologram: each part contains the whole. Ascott states it most fully in the title essay (ch. 20): "Each individual computer interface is an aspect of a telematic unity, such that to be in or at any one interface is to be in the virtual presence of all the other interfaces throughout the network of which it is a part. This might be defined as the 'holomatic' principle in networking." (`Directly stated`) The reason he gives is informational, not metaphorical: "all the data flowing through any access node of the network are equally and at the same time held in the memory of that network" — accessible, via cable or satellite, "from any part of the planet at any time of day or night." (`Directly stated`) Presence is thereby decoupled from physical location and synchronous time: users "do not need to be in the same place at the same time" to be co-present in the network.

The figure is prefigured two chapters earlier in *Gesamtdatenwerk* (ch. 18), where Ascott describes the interface as "an aspect of a holographic unity. To be in or at any one interface is to be with all the interfaces throughout the network of which it is a part." (`Directly stated`) The shift from "holographic unity" to the named "holomatic principle" marks Ascott settling the coinage; the underlying claim — part-as-whole, interface-as-network — is constant across both. By *Telenoia* (ch. 23) and *The Architecture of Cyberception* (ch. 27) the phrase has stabilized as Ascott's standard term, where he glosses it as "each individual interface is an aspect of a telematic unity" and ties it to the lived perceptual faculty of [[Cyberception]]: "Cyberception gives us access to the holomatic media of the cybernet." (`Directly stated`, ch. 27)

The holomatic principle is the technical underwriting of Ascott's larger claims. It is what makes the [[The Interface as Doorway|interface]] not a window onto a finished reality but "a doorway to undecidability, a data space of semantic and material potentiality" — content is created at the interface rather than received, then "reinserted, transformed by the process of interaction, back into the network" for transformation at the interfaces of other users elsewhere. (`Directly stated`, ch. 20) Ascott demonstrated it concretely at the 1986 Venice Biennale's "Planetary Network" project, which "had the effect of pulling the exhibition from its rather elite, centralised, and exclusive domain in Venice and stretching it out over the face of the globe," so that "the data flow at any one point was the data flow at all other points in the system." (`Directly stated`, chs. 20, 23) The principle thus grounds [[Network Consciousness]] and a distributed, participatory model of [[Telematic Art]]: a single shared planetary memory in which every node holds the whole. (`Interpretive synthesis`)

## Authors and Variants

- **[[Roy Ascott]]** — coins the term. Earliest form is "holographic unity" of the interface (ch. 18, *Gesamtdatenwerk*); named as the "holomatic principle" in the title essay (ch. 20) and reused in *Telenoia* (ch. 23) and *The Architecture of Cyberception* (ch. 27). The constant claim: because the whole network's data are held simultaneously at every node, to be at one interface is to be in the virtual presence of all interfaces. In ch. 27 it becomes the perceptual content of [[Cyberception]] ("the holomatic media of the cybernet").

## Related Concepts

- [[The Interface as Doorway]] — the holomatic principle is what the interface *is when networked*; it makes each interface an aspect of the telematic unity rather than a discrete terminal.
- [[Network Consciousness]] — the holomatic principle is the structural basis for Ascott's claim that the network constitutes a distributed planetary consciousness.
- [[Cyberception]] — Ascott's 1994 perceptual faculty; in ch. 27 cyberception is precisely what "gives us access to the holomatic media of the cybernet."
- [[Telematic Art]] — the art practice the principle enables: participatory, distributed, with meaning produced at and across interfaces rather than fixed in an object.
- [[Reciprocal Communication]] — the interactive two-way flow whose planetary reach the holomatic principle describes; data inserted at one node is transformed at others.
- [[Distributed Cognition]] — the holomatic network is a substrate for cognition spread across nodes rather than localized in a single mind or place.
- [[The Global Village]] — McLuhan's electric simultaneity, which Ascott extends from broadcast toward a literal shared data-memory in which every node holds the whole.
- [[Virtuality]] — "to be in the virtual presence of all the other interfaces": holomatic co-presence is a virtual, not corporeal, presence.

## Opposing Concepts

- [[Material vs Immaterial]] — the holomatic principle is Ascott's most enthusiastically immaterialist move: presence, memory, and artwork are relocated from material object and physical place into the simultaneously-shared data-memory of the network. It is the same distributed, networked, surveillance-capable condition that other vault authors (Han) diagnose as domination rather than embrace as liberation.

## Questions It Raises

- If all data are "equally and at the same time held in the memory of that network," what becomes of authorship, originality, and the singular art object — is content ever anyone's, or only ever the network's?
- Holomatic co-presence dissolves the requirement of same-place/same-time. Does this abolish distance and the Other, or merely re-mediate them — and is the result the telematic embrace or the surveillant capture of every node by every other?
- The principle treats the part as containing the whole. Does a real network actually distribute its memory holographically, or is "holomatic" an aspirational metaphor that hides centralized storage and control behind a rhetoric of ubiquity?
- If to be at any interface is to be present at all of them, what counts as presence at all — and what is lost when corporeal, located presence is recoded as virtual presence?
