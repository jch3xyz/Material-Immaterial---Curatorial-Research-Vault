# Art from object to system to network

The art spine runs from object to system to network. Burnham watches modern sculpture leave the pedestal and become a system; Lippard watches the art object dematerialize into idea and information; Ascott watches the artwork migrate off the object entirely and become the network itself. This pathway is useful curatorially for staging the lineage of post-object art — systems esthetics, conceptual art, and telematic/new-media practice as one continuous migration — and for tracing how the Material/Immaterial axis plays out specifically within art history rather than philosophy or cybernetics.
