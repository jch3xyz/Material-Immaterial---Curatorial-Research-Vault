# About — How We Became Posthuman

**Author:** N. Katherine Hayles  (University of Chicago Press, 1999)
**Source PDF:** 366-page scanned book (400 dpi bilevel JBIG2 images). The embedded text layer was low-quality OCR with systematic character corruption, so every page was **re-OCR’d from the page images with Tesseract 5** for this extraction.

## Structure
- Acknowledgments, Prologue, 11 numbered chapters, and endnotes (Notes), one markdown file each.
- Chapter headings and titles regenerated from the PDF outline; decorative chapter-title pages and running headers were dropped.
- Several chapters open with an **epigraph** (quotation + attribution), rendered as a blockquote.
- Endnotes are grouped by chapter (`## Chapter X`) with numbered items.

## Extraction caveats
- Text was produced by fresh OCR; despite high accuracy, occasional OCR errors remain (e.g., a stray character for a glued word, or a misread letter). This is inherent to OCR of a scanned book.
- **Endnote reference markers** in the body (superscript numbers in print) frequently OCR as a stray character (e.g. `.`, `!`, `’`, `*`) attached to the preceding word; these were left in place rather than guessed at.
- Line-break hyphenation was rejoined using a dictionary plus a vocabulary built from the book; genuine compounds (e.g. *self-organization*, *post-human*) are preserved.
- Figures/diagrams are images; where OCR produced only noise, a `*[figure or diagram...]*` marker was inserted.
- `source_pages` refers to **PDF page numbers** (the scan sequence), not the book’s printed page numbers.
- Index excluded: it maps to print page numbers and is low-value for a text vault.
