## 2026-06-17 — Synthesis-essay framework upgraded (reads as essay, not outline)

The user judged the synthesis-essay template too simplistic — it produced an **outline** (labelled `## Central Question` / `## Thesis` / `## Section N`, plus an inline `` `Directly stated` `` tag after nearly every quote) rather than an **essay**. - **Design — two layers (the read vs. the apparatus).** The reading layer is continuous, argued prose: no functional headings (the thesis is a plain sentence, usually closing the opening movement; movements carry evocative sentence-case subheads or a `* * *` break, never "Section N"); in-text attribution ("as Hayles puts it") carries the citation; an optional `## Coda` names the curatorial payoff. The provenance apparatus moves **below a horizontal rule**, exactly as a journal prints footnotes/acknowledgments: a `## A note on method` paragraph + a `## Notes` block of Obsidian footnotes.
- **Provenance fully preserved (the hard rules survive a clean read).** Citation mechanism = **Obsidian `[^n]` endnotes** (user-chosen). Each note carries the `[[raw/Author - Book/Chapter]]` link + (if not quoted in full in the body) the verbatim passage + a compact status/confidence tag (`Directly stated, high` / `Strongly implied, medium` / `Interpretive synthesis, medium` / `Speculative connection, low`). So: verbatim quotes ✓, a confidence label on every load-bearing claim ✓, reversible compression to the raw chapter ✓, no invented citations ✓ — the apparatus just no longer pollutes the prose. The footnotes **are** the claim audit; a standalone `## Claim ledger` table is now optional, dropped by default.
- **Template** (`_system/templates/synthesis_essay.md`) rewritten as a **craft brief + copy-paste skeleton** (frontmatter gains a `deck`/standfirst field): the cardinal rule ("nothing in the reading layer announces its function"), the felt shape (opening → movements → close), voice/evidence/length guidance (~1,200–3,000 words, semi-academic), and the relocated-provenance spec.
