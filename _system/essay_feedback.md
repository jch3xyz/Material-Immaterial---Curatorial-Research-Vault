# Essay Feedback — raw notes, one section per essay

This is a **log, not a law.** Write freely and fast here — reactions, half-formed hunches,
things that bugged you, things that landed. Nothing you write here changes the next essay
directly; a generation agent never reads this file as rules. It is read only by the
**distillation agent**, which watches for patterns *across* essays before anything is promoted
into `essay_principles.md`. That gap is what lets you react quickly without causing an
overcorrection — so don't self-censor here.

**Tags** (optional, help the distiller read intent — put them inline next to a note):
- `[fix]` — do less/more of X next time
- `[keep]` — this worked; do more of it (positive feedback matters as much as correction)
- `[one-off]` — specific to this essay's topic; do NOT generalize into a principle
- `[promote-now]` — you're sure; make this a principle immediately (enters provisional, still on probation)

**Convention:** each new essay gets a new `##` header here when it's generated. Date your notes
if you like; the distiller uses recency and recurrence.

The full loop — tiers, promotion gate, who distills — is documented at the top of
`essay_principles.md`.
